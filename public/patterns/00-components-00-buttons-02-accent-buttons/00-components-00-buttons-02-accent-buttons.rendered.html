Accent Buttons

Accent buttons are to be used where there are multiple buttons on the screen and we want to call attention to the most important action on the screen. Only one accent button should be visible on the screen at a time.

State: Default
State: Hover
State: Pressed
  • HTML
  • Accent Button LESS
  • Base Button LESS
        
          
        
      
        
          .button {
            ...
            &_accent {
              color: @color-white;
              background-color: @color-orange;
              border: 2px @color-orange solid;
              .box-shadow;

              &:hover {
                background-color: darken(@color-orange, 10%);
                border: 2px darken(@color-orange, 10%) solid;
                transition: all .3s ease;
              }
              &:active, &:visited, &:focus {
                background-color: lighten(@color-orange, 10%);
                border: 2px lighten(@color-orange, 10%) solid;
                transition: all .3s ease;
              }
            }
          }
        
      
        
          .button {
            border-radius: 2px;
