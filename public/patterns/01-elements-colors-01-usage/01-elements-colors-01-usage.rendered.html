Color Usage

Gale Blue

Primary Blue is to be used as a background color (button, banner, etc.) and for text links on light backgrounds.

Example
@color-blue-primary
#006298
Text Example
Button Link

Dark Blue is to be used as a background color or title text

Example
@color-blue-dark
#003865

Neutral Colors

White is the default background color. Should be used as the text color for any text on the primary blue, dark blue, or secondary colors.

Example
@color-white
#ffffff
Text Example

Light Gray is the background color that should be used when the Card component is being used. Should never be used as a text color.

Example
@color-gray-light
#f2f2f2
Text Example

Medium Gray should be used when subtle accent colors are needed to differentiate areas on the White or Light Gray background. Should never be used as a text color.

Example
@color-gray-medium
#d6d6d6
Text Example

Dark Gray should be used as a background color only when White and Light Gray are already in use as background colors. Should never be used as a text color.

Example
@color-gray-dark
#667780
Text Example

Darker Gray should be used as a background color for lightboxes and used at 80% opacity. Should be used for drop shadows at 30% opacity.

Example
@color-gray-darker
#002133
Text Example

Black is the default color for text.

Example

#000000
